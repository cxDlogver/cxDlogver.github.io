Generator

1. Generator描述

Generator（生成器） 是 JavaScript 中一种可暂停、可恢复执行的函数。它通过 function* 定义，并使用 yield 将函数的执行过程拆分为多个阶段。调用 Generator 函数并不会立即执行函数体，而是返回一个迭代器对象；每次调用迭代器的 next() 方法，函数才会从上一次暂停的位置继续执行，直到遇到下一个 yield 或执行结束。

在执行过程中，yield 用于产出当前阶段的值并暂停函数，而下一次 next(value) 传入的参数会作为上一次 yield 表达式的返回值，从而实现外部与函数内部之间的双向数据传递。当函数执行到 return 或自然结束时，Generator 终止，并返回 { done: true } 状态。

从本质上看，Generator 并不是异步方案，而是一种流程控制工具。它常用于实现自定义迭代器、状态机，以及在 async/await 出现之前配合 Promise 执行器控制异步流程。async/await 可以被视为基于 Generator 思想的自动化、工程化实现。
